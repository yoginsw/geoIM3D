# Upstream COPR packaging archive

이 디렉터리는 MIT Fork Attribution과 Upstream 참고를 위해 보존합니다.

- geoIM3D의 승인된 배포 채널이 아닙니다.
- COPR Publish Workflow와 API Token 사용 경로는 제거됐습니다.
- Upstream `geolibre` Spec은 geoIM3D Package로 게시하거나 재사용하지 않습니다.
- 향후 JBT 소유 Package ID, Code signing, Runtime 검증, 독립 Release 승인이 완료된 경우에만 별도 요구사항으로 재개합니다.
