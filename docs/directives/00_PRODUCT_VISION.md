# geoIM3D 제품 비전

## 제품 정의

**지오아임3D(geoIM3D)** 는 JBT가 제공하는 건축·토목·부동산·환경·안전 분야의 **실감형 3D 공간 플랫폼**이다.

GeoLibre 2.1.0의 GIS 기반을 활용하되, 사용자 경험과 배포 Identity를 JBT 제품으로 재구성하고 VWorld와 국내 공간 업무 기능을 강화한다.

## 대상 사용자

- 건축·건설 프로젝트 관리자와 설계자
- 토목·측량·공간 분석 담당자
- 부동산 개발·입지 분석 담당자
- 환경·안전·재난 분석 담당자
- 2D/3D 공간 데이터를 통합 검토하는 공공·민간 실무자

## 핵심 가치

1. 2D와 3D 공간정보의 단일 작업환경
2. CAD/BIM/GIS/Point Cloud/실감 데이터를 함께 시각화
3. Browser와 Windows Desktop에서 동일한 프로젝트 경험
4. Offline에서도 핵심 데이터와 분석 기능 사용
5. VWorld 기반 국내 공간정보 연계
6. AI 기반 공간 분석과 반복 업무 자동화
7. 기존 GeoLibre Plugin 생태계 호환

## 범위 밖

초기 geoIM3D 1.0에서 기본 UI에 노출하지 않는다.

- Jupyter Notebook
- Python Console
- Collaboration/Chat
- Field Collection

기능 코드는 제거하지 않고 재활성화 가능한 상태로 유지한다.
